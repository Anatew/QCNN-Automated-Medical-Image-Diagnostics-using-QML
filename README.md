QCNN for Medical Image Classification
Overview:
This project implements a Quantum Convolutional Neural Network (QCNN) to classify facial images
for detecting disorders like Parkinson’s and Autism, and compares it with a classical CNN.
Features:
- QCNN model using quantum circuits
- CNN model for comparison
- Medical image classification
- Performance evaluation
Tech Stack:
Python, Qiskit, TensorFlow/PyTorch, NumPy, Matplotlib
Project Structure:
dataset/
cnn_model/
qcnn_model/
notebooks/
How to Run:
1. Clone the repository
2. Install dependencies
3. Run CNN and QCNN models
Output:
- Classification results
- Accuracy comparison
Note:
QCNN results are based on simulation.
Future Work:
- Real quantum hardware implementation
- Hybrid QCNN models
- Improved accuracy
